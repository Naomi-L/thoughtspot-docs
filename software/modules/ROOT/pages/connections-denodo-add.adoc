Follow these steps to connect to Denodo databases:

include::partial$connections-nav1.adoc[]

. In the *Choose your data warehouse* interface, specify the fields:
+
xref:connections-denodo-reference.adoc#connection-name[Name your connection]:: For example, _MyDenodoDatabase_
xref:connections-denodo-reference.adoc#connection-description[Describe your connections]:: The optional description of the database
Choose your data warehouse:: Select the Denodo tile
+
image::connection-databricks-add.png[Add Denodo connection]

. Click *Connect*.

. In the *Specify connection details* interface, enter the connection details for your Denodo data source.
+
image::connection-databricks-specify-details.png[]
+
xref:connections-denodo-reference.adoc#host[Host]:: Denodo server host name without the protocol.
xref:connections-denodo-reference.adoc#http-path[HTTP Path]:: Path to the host in HTTP format
xref:connections-denodo-reference.adoc#user-id[User]:: The user account name
xref:connections-denodo-reference.adoc#password[Password]:: The password of the user account
+
Refer to the xref:connections-denodo-reference.adoc[Denodo connection reference] for more information on each of the specific attributes you must enter for your connection.
