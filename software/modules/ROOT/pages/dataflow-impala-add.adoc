= Add an Impala connection
:last_updated: 7/6/2020
:experimental:
:linkattrs:
:page-aliases:



You can add a connection to an Impala database using ThoughtSpot DataFlow.

Follow these steps:

include::partial$add-database-connection.adoc[]

. After you select the Hive *Connection type*, the rest of the connection properties appear.
+
See xref:dataflow-hive-reference.adoc#connection-properties[Connection properties] for details, defaults, and examples.
+

xref:dataflow-impala-reference.adoc#dataflow-impala-conn-connection-name[Connection name]::  Name your connection.
xref:dataflow-impala-reference.adoc#dataflow-impala-conn-connection-type[Connection type]::  Choose the Impala connection type.
xref:dataflow-impala-reference.adoc#dataflow-impala-conn-host[Host]::  Specify the hostname or the IP address of the Impala system.
xref:dataflow-impala-reference.adoc#dataflow-impala-conn-port[Port]:: Specify the port associated to the Impala system.
xref:dataflow-impala-reference.adoc#dataflow-impala-conn-authentication[Authentication]:: Select the type of Authentication being enabled.
xref:dataflow-impala-reference.adoc#dataflow-impala-conn-default-realm[Default realm]:: Realm of the Impala host. Appears when Authentication Type is "Kerberos".
xref:dataflow-impala-reference.adoc#dataflow-impala-conn-kerberos-host-name[Kerberos host name]:: Kerberos service principal name of the Impala server. Appears when Authentication Type is "Kerberos".
xref:dataflow-impala-reference.adoc#dataflow-impala-conn-kerberos-service-name[Kerberos service name]:: Kerberos service principal name of the Impala server. Appears when Authentication Type is "Kerberos".
xref:dataflow-impala-reference.adoc#dataflow-impala-conn-user[User]:: Specify the user id that will be used to connect to the Impala system. This user should have necessary privileges to access the data in the databases. +
Appears when Authentication Type is "Kerberos" and "LDAP".
xref:dataflow-impala-reference.adoc#dataflow-impala-conn-password[Password]:: Specify the password for the User. Appears when Authentication Type is "LDAP".
xref:dataflow-impala-reference.adoc#dataflow-impala-conn-jdbc-options[JDBC options]:: Specify the options associated with the JDBC URL.


. Click *Create connection*.
